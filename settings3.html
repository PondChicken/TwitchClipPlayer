SLOBS / OBS Setup

Now that you have your custom URL, you need to add a browser source in you SLOBS or OBS application.

Please note the following IMPORTANT points:

  1. Local file should be unchecked. This is needed so that the URL will support URL parameters
  2. Paste your URL in the URL input
  3. Try to ensure that the width and height are the same aspect ratio as your video. It's best to set the values here rather than trying to resize the video with your mouse in the scene directly
  4. It's best to check the options "Shutdown source when not visible" and "Refresh browser when scene becomes active"
