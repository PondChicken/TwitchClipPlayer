Credits

This project was built on top of some excellent work from others developers:

  • ShaderWave's Random Twitch Clip Player:
    https://github.com/ShaderWave/RandomClipPlayer/releases/tag/v1.1
  • Mehdi's CSS only Scanlines Overlay:
    https://codepen.io/meduzen/pen/zxbwRV

This project takes ShaderWave's original project and adds the following features:

  • The ability to pass configuration parameters from the URL
  • Added the scan lines overlay option
  • Added between video static option
  • Added the option to filter the random clips to a set of named twich users
  • Created a Simple UI (that you are looking at now) to allow a user to easily configure the application without any knowledge of javascript and HTML
